마크다운

What is Markdown ?

마크다운 : 일반 텍스트 기반의 경량 마크업 언어(mark language)다.
마크업언어 : 태그 등을 이용하여 문서나 데이터의 구조를 명기하는 언어의 한 가지이다.

History

마크다운 역사 : John Gruber는 2004년에 문법(syntax)면에서 Aaron Swartz와 중대한 협업(collaboration)을 통해 마크다운 언어를 만들었으며,
사람들이 읽기 쉽고 쓰기 쉬운 플레인 텍스트 포맷을 사용하여 쓸 수 있으면서 구조적으로 유효한 XHTML(또는 HTML)로 선택적 변환이 가능하게 하는 것이 목표이다

Syntax

  • <제목(Headers)>
# 대제목  
## 중제목  
### 소제목

| :------------ | :-----------: | -------------------: |
| First row     | Data          | Very long data entry |
| Second row    | **Cell**      | *Cell*               |
| Third row     | Cell that spans across two columns  ||
